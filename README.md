## Setup
```bash
npm install
npm run setup
```

## Compile
```bash
npm run compile
```

## Test in browser
```bash
npm run dev-server
```

## Build Cordova project
```bash
npm run build-cordova
```

## Open Cordova project in XCode
```bash
npm run xcode
```

## Re-convert wav files
```bash
npm run convert-audio
```
